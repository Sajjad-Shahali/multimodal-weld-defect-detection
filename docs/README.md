# Documentation

This folder contains lightweight project documentation for the WeldFusionNet repository.

The main goal of `docs/` is to explain how the pipeline outputs are organized without bundling large generated artifacts into version control.

## Included files

- `outputs_map.md` maps each pipeline step to its generated files under `output/`.

## Notes

- Large reports, model checkpoints, and full datasets are intentionally not stored here.
- Use this folder for project notes, output references, and any future GitHub Pages or demo-facing documentation.
- For the full project overview, start from the repository root `README.md`.
